What is CFIHOS?


The Capital Facility Handover Standard, or CFIHOS (pronounced see-foss), is a Joint Industry standard developed by major facility constructors, suppliers and owners.

Visit the official site here, read more about it here, and watch a video presentation here.

What is Spectrum CFIHOS?


It's a free set of API engines that you can run in a collection of Docker containers.

Created with a Microservice architecture, it provides an intuitive REST based API to create and query all data objects in the CFIHOS standard.

How do I get it?


Taking the API for a test drive is easy.

Ensure you have Docker installed on your machine, and follow the instructions on our DockerHub page.

How does it scale?


Spectrum CFIHOS is built with a Miroservice architecture on a containerized platform (Docker).

It is completely 'cloud native'. That means is can be easily and reliably deployed and scaled using an orchestration platform like Kubernetes.

How 'free' is it?


It's free... as in FREE.

The Spectrum API engine image is hosted on DockerHub. From there is it free to download and run on your own machine, or on your own infrastructure.

All we ask is that you consider providing feedback on your experience so we can improve the product.

That's it! No catch.
